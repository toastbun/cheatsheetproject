HTML Basic Tag Functions

Common Tags

Tag Name Definition
<h1> Header 1 Creates the largest style of header text. Number can be changed for smaller headers.
<p> Paragraph Allows for paragraphs of text to show on webpage.

Special Tags

Tag Name Definition
<embed> Embed Used to embed an image, video, or sound file into your webpage.
<link> Link Allows you to link an external source to an HTML document, among other things.

Important & Boilerplate Tags

Tag Name Definition
<html> HTML A required tag that all of your head, body etc should be inside.
<!DOCTYPE HTML> Document Type Tells the browser that your document is written in HTML, and it should be interpretted that way.
